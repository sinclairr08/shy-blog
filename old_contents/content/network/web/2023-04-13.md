# XMLHttpRequest

## XMLHttpRequest

- 자바스크립트에서 서버에서 데이터를 가져오기 위해 사용되는 자바스크립트 객체. AJAX에서 주로 사용
- 서버에서 화면의 새로고침 없이 데이터를 가져올 수 있음
- 이름에서 보이는 것과 달리 XML 뿐만 아니라 모든 형태의 데이터를 가져올 수 있음
- 개발자가 사용하는 경우는 드물며, 브라우저에 내장되어 있음
  - 최근에는 Fetch, Axios등을 이용해 서버와의 통신을 구현

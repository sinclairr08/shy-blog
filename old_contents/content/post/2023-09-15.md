# NFS

## NFS

- Network File System의 약자로, 분산 파일 시스템 프로토콜
- 서버가 여러 클라이언트에게 파일 및 디렉토리를 공유
- 클라이언트끼리 파일을 전송하지 않아도, 공유할 수 있다는 장점 존재
  - 여러 컴퓨터에서 협업하기가 쉬움
- 클라이언트 - 서버 간의 통신에 RPC 통신을 기반으로 수행

# Tomcat

## Tomcat

- Apache 재단에서 만든, 서블릿 컨테이너 및 HTTP 처리 엔진만 있는 WAS
- 자바 서블릿을 이용해 요청을 자바로 처리하고 응답
- Apache 2.0 License를 채용한 오픈 소스 SW
- Tomcat 5.5 버전부터는 Apache의 정적 컨텐츠를 처리하는 web server 기능이 추가됨. 이후로 Apache Tomcat이라고 불림

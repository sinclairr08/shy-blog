# Hadoop

## Hadoop

- 하둡이란 High Availability Distributed Object Oriented Platform의 약자
  - 말 그대로 객체 지향적 작업을 분산해서 수행함으로 인해 고가용성을 확보
- 오픈 소스이며, Java 기반의 소프트웨어 플랫폼
  - 애플리케이션용 데이터 처리 및 스토리지 관리
  - 서버를 확장하기 쉬움
- 하둡 데이터는 HDFS에 저장됨
  - 비구조적이므로 RDBMS와 다름

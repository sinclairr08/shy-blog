# Bagging Boosting

## Bagging Boosting

- Bagging
  - 데이터를 여러 번 샘플링한 후, 개별 모델을 학습하는 방식
  - 이후 각 모델의 결과를 투표하거나 평균을 내서 결과를 취합
  - 각 개별 모델은 독립적으로
  - Overfitting의 위험성 존재
- Boosting
  - 오분류된 데이터에 더욱 큰 가중치를 주어 학습하는 방식
  - 약한 모델부터 시작해서, 순차적으로 모델의 틀린 부분을 학습해 나감

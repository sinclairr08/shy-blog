# Sequential Access Random Access

## Sequential Access

- 데이터에 접근하는 방식 중 하나로, 순서가 정해진 데이터를 순차적으로 접근하는 방식
- 데이터 갯수 및 위치에 접근 속도가 영향을 받으며, $O(N)$의 시간 복잡도를 가짐
- Linked list를 접근할 때 해당됨

## Random Access

- 데이터의 주소를 통해 접근하는 방식
- 데이터 갯수나 위치에 접근 속도가 영향을 받지 않으며, $O(1)$의 시간 복잡도를 가짐
- Array를 접근할 때 해당됨

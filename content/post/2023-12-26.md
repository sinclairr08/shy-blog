# Python Lambda

## Python Lambda

- Python에서 익명 함수를 사용할 때, lambda라는 키워드를 사용
- lambda 인자: 표현식 형태로 사용
  - 인자들은 각각 comma로 구분
  - 표현식은 object를 반환
- 예시: `lambda x, y: x ** y`
- map, filter 등 함수 자체를 인자로 넘겨야하는 메서드의 경우 주로 사용

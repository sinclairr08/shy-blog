# Fastapi

## Fastapi

- 파이썬으로 빠르게 API를 빌드할 수 있는 웹 프레임워크
  - python 3.6+ 필요
- 특징
  - Node 및 Go에 비견될 정도로 빠름
  - 배우기 쉬우며, 코드 작성을 빠르게 가능
  - 버그가 적으며, 직관적임
- `pip install fastapi`로 설치 가능
- node와 비슷하게 `@app.get("/")` 와 같은 형태로 api를 만들 수 있음

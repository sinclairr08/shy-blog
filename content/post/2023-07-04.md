# Private Method Test Code

## Private Method Test Code

- Private 메소드의 테스트 코드는 작성하지 않는 것이 좋음
  - Private method를 테스트 해야 한다면 책임이 잘못 할당되거나 설계가 잘못되었다는 뜻
  - 리팩토링을 해야 한다는 신호
- 그러나 꼭 private 메소드를 테스트 해야 하는 경우에는 reflection 등을 이용해 테스트 가능

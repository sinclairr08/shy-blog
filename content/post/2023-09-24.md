# Modin (Pandas)

## Modin (Pandas)

- Pandas에서 코드 교체 없이 성능을 업그레이드 할 수 있는 라이브러리
  - 이를 drop-in replacement라고 함
- 멀티 코어를 사용하게 함으로 인해, pandas의 성능을 높임
  - `read_csv` 등의 메소드 속도가 빨라짐
- `pip install modin[all]`을 통해 설치
- `import modin.pandas as pd` 로 import하는 부분만 바꾸면 됨

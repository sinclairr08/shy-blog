# npm 및 설치

## npm 및 설치

- Node Package Manager의 약자로, 자바스크립트 패키지 매니저
  - 세계에서 가장 큰 소프트웨어 저장소이기도 함
- 설치 및 관리에 CLI를 사용
- Node.js를 설치할 때 같이 설치되며, node.js 설치에는 nvm을 사용

## 설치 예시

```sh
nvm install 16
nvm use 16
npm -v
```

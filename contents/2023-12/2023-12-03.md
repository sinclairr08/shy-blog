# Null Device

## Null Device

- OS에서 쓰기가 되는 모든 데이터를 버리지만, 쓰기 작업 자체는 성공이라고 보고하는 장치를 의미
- Unix 계열 시스템에서는 `/dev/null` 이며, 각 운영체제마다 다른 이름을 가짐
- `[COMMAND] > /dev/null 2>&1` 와 같이 출력 파일을 /dev/null로 지정하는 경우, 출력 자체를 모두 없앤다는 의미가 됨
